# Proyecto Cliente/Servidor - Avance 2

## Descripción
Sistema desarrollado en Java que permite la gestión de usuarios y rutinas de entrenamiento mediante una interfaz gráfica.

## Funcionalidades
- Inicio de sesión
- Registro de usuarios
- Creación de rutinas
- Visualización de ejercicios

## Estructura del proyecto
- src/app → clase principal
- src/modelo → lógica del sistema
- src/vista → interfaz gráfica

## Cómo ejecutar
1. Abrir el proyecto en NetBeans
2. Ejecutar la clase Main.java
